# CTS Group Website

A modern, responsive multi-page website for CTS Group Construction, built with pure HTML5, CSS3, and vanilla JavaScript.

## Features

- **5 Complete Pages**: Home, About, Projects, Team, and Careers
- **Responsive Design**: Mobile-first approach with hamburger menu
- **Modern UI**: Smooth animations, hover effects, and professional styling
- **SEO Ready**: Proper meta tags and semantic HTML
- **Fast Loading**: No heavy frameworks, optimized CSS
